Задача «Минимальная цена из списка» На странице есть список товаров с ценами. Пользователь может добавлять новые цены через форму (input + кнопка «Добавить цену»). Все цены отображаются в списке. Есть кнопка «Найти минимальную цену». При нажатии на неё выводится самая маленькая цена из всех добавленных с помощью Math.min.
