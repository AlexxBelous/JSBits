Задача «Лучший результат среди учеников» На странице есть форма для ввода оценок учеников. Пользователь может добавлять оценки (числа от 0 до 100). Все оценки отображаются в списке. Есть кнопка «Показать лучшую оценку». При нажатии на кнопку выводится самая высокая оценка с помощью, а также имя ученика с этой оценкой.
